# 智能笔记助手 PRD

**文档版本**：V1.0  
**创建日期**：2026-04-09  
**产品名称**：智能笔记助手  
**目标用户**：职场人士  
**预计上线时间**：3个月后

---

## 1. 产品概述

### 1.1 产品背景

职场人士每天需要处理大量会议，整理会议纪要是一项耗时且繁琐的工作。传统手动记录方式效率低下，容易遗漏关键信息。智能笔记助手旨在通过AI技术，帮助职场人士高效完成会议纪要的整理工作。

### 1.2 产品目标

帮助职场人士快速、准确地记录和整理会议内容，提升信息整理效率300%以上。

## 2. 核心功能规划

### 2.1 功能优先级矩阵

| 功能 | 优先级 | 预计工期 | 依赖关系 |
|------|--------|----------|----------|
| 语音转文字 | P0（最高） | 6周 | 无 |
| 自动摘要 | P1 | 4周 | 语音转文字 |
| 标签分类 | P2 | 3周 | 自动摘要 |

### 2.2 功能详细说明

#### P0 - 语音转文字

**功能描述**：用户通过App录制会议音频，系统自动将语音转换为文字。

**用户场景**：
- 会议进行中，打开App实时录音
- 录音完成后，一键转换为文字稿
- 支持中英文混合识别

**技术要求**：
- 语音识别准确率 ≥ 95%（标准普通话）
- 单次录音最长支持2小时
- 转换延迟 < 录音时长的30%

**关键需求**：
- 离线录音保存，断网不丢失
- 录音质量自适应调整
- 支持后台录音

#### P1 - 自动摘要

**功能描述**：系统自动分析会议文字内容，生成结构化摘要。

**输出格式**：
- 会议主题
- 关键决议（3-5条）
- 行动项（待办事项）
- 重要时间节点

**关键需求**：
- 摘要生成时间 < 文字总量的10%
- 支持用户手动编辑摘要
- 摘要长度自适应（短会议简短，长会议详细）

#### P2 - 标签分类

**功能描述**：系统自动识别会议内容并打上标签，支持用户自定义标签体系。

**预置标签类别**：
- 会议类型：周会、月会、项目会议、一对一
- 主题标签：技术、产品、市场、运营、人事
- 情绪标签：正面、负面、中性

**关键需求**：
- 每篇会议记录至少打1个标签
- 支持多标签组合
- 标签管理（增删改查）

## 3. 用户旅程

```
用户打开App
    │
    ▼
点击「新建笔记」→ 开始录音
    │
    ▼
会议进行中（后台录音）
    │
    ▼
点击「结束」→ 自动转写为文字
    │
    ▼
系统自动生成摘要 + 标签
    │
    ▼
用户浏览、编辑、确认
    │
    ▼
保存至笔记列表
```

## 4. 第一阶段范围（移动端App）

### 4.1 MVP功能列表

1. **语音录制**
   - 开始/暂停/结束录音
   - 录音时长显示
   - 录音列表（本地存储）

2. **语音转文字**
   - 一键转换
   - 转写进度显示
   - 原始文字稿展示

3. **基础笔记管理**
   - 笔记列表（按时间排序）
   - 笔记搜索
   - 笔记删除

### 4.2 非功能性要求

- iOS 14+ / Android 10+ 支持
- App安装包 < 80MB
- 冷启动时间 < 3秒
- 文字转写API响应时间 < 10秒（5分钟内音频）
